# Cloud-Fire-Plataforma-de-Armazenamento-em-Nuvem-
Cloud Fire é uma plataforma demonstrativa de armazenamento em nuvem inspirada no MediaFire, criada para fins educativos e portfólio. O sistema permite enviar, listar, visualizar e baixar arquivos armazenados em serviços compatíveis com Amazon S3, utilizando o Wasabi Hot Cloud Storage
